# intake-form

Application for free legal services intake form.

## Contributing

**Important**: The `main` branch is protected. All changes must be made through pull requests.

Please see [CONTRIBUTING.md](CONTRIBUTING.md) for detailed guidelines on how to contribute to this project.

### Quick Start for Contributors

1. Create a new branch from `main`
2. Make your changes
3. Push your branch and create a pull request
4. Wait for review and approval before merging

See [`.github/BRANCH_PROTECTION.md`](.github/BRANCH_PROTECTION.md) for information about branch protection settings.

## Branch Protection

This repository implements branch protection to ensure code quality:

- ✅ All changes require pull requests
- ✅ Pull requests require code owner approval  
- ✅ Automated checks must pass before merging
- ✅ Direct commits to `main` are prevented

For more details, see the documentation in the `.github` directory.
